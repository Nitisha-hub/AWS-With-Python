# ☁️ Amazon S3 (Simple Storage Service)

## 📌 What is S3?

Amazon S3 is a cloud storage service that allows you to store and retrieve data from anywhere on the internet.

---
 
## 🚀 Uses

* Store images, videos, and files
* Backup and restore data
* Host static websites
* Store logs and big data

---

## ⚙️ Key Features

* Unlimited storage
* Highly durable (99.999999999%)
* Secure (IAM policies, bucket policies)
* Scalable
* Pay-as-you-go pricing

---

## 🧠 Important Concepts

### 🔹 Bucket

A container to store objects (files)

### 🔹 Object

The actual file stored in S3

### 🔹 Key

Unique name of the object in a bucket

### 🔹 Region

Location where bucket is created

---

## 🔐 Security

* IAM roles and policies
* Bucket policies
* Public/Private access control

---

## 🔄 Storage Classes

* S3 Standard
* S3 Intelligent-Tiering
* S3 Glacier (for archive)

---

## 📸 Workflow

User → Upload File → S3 Bucket → Store & Access Anytime

---

## 💡 Example

Uploading an image to S3 bucket and accessing it via URL.
