# Amazon EC2 (Elastic Compute Cloud)

## 📌 What is EC2?

EC2 is a web service that provides resizable virtual servers in the cloud.

---

## 🚀 Uses

* Hosting websites
* Running applications
* Testing and development
* Backend servers

---

## ⚙️ Key Features

* Virtual machines (instances)
* Scalable (increase/decrease capacity)
* Pay-as-you-go pricing
* Multiple instance types

---

## 🔑 Important Concepts

* AMI (Amazon Machine Image)
* Instance Types (t2.micro, etc.)
* Key Pair (for login)
* Security Groups (firewall)

---

## 💡 Example

Launching an EC2 instance to host a website.
